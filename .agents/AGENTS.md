# Rules

- 所有涉及中文的內容、翻譯、字典、標籤或文案，一律使用台灣正體中文（繁體中文，zh-TW），嚴禁使用簡體中文（zh-CN）。
- 回覆使用者時，一律使用台灣正體中文，不可使用簡體中文或非台灣習慣之用語。
- 嚴格遵守台灣資訊科技用語，嚴禁使用中國大陸習慣用語。例如：
  * 使用「程式碼」而非「代碼」
  * 使用「專案」而非「項目」
  * 使用「快取」而非「緩存」
  * 使用「伺服器」而非「服務器」
  * 使用「檔案」而非「文件」
  * 使用「網頁重新整理/重整」而非「刷新」
  * 使用「螢幕擷取畫面/螢幕截圖」而非「截屏」

## 版本升級同步規範
- 每當升級 App 版本號時，必須**同步更新**以下三個檔案中的版號與說明字串，避免版號被覆寫或快取失效：
  1. **`service-worker.js`**：更新 `CACHE_NAME` 與 `ASSETS` 陣列中靜態資源的 `?v=XX` 版本參數。
  2. **`index.html`**：更新資源引用的 `?v=XX` 版本參數，以及 `#app-version-label` 標籤的預設文字。
  3. **`app.js`**：更新 `TRANSLATIONS` 字典中所有語系（`zh-TW`、`en`、`ja`、`sv`、`ne`）下的 `"app_version"` 對照字串。
